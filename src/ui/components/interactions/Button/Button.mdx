# ボタンコンポーネント

ボタンコンポーネントは、アクションのトリガーに使用されます。

## 特徴

- 複数のバリアント（primary, secondary, accent, outline, ghost, link）
- 5つのサイズ（xs, sm, md, lg, xl）
- 角丸のカスタマイズ
- アイコンのみのボタンサポート
- フルワイド表示オプション

## インポート方法

```tsx
import { Button } from "@/ui/components/interactions";
```

## 使用例

### 基本的な使い方

```tsx
// デフォルトのボタン (primary, md)
<Button>クリック</Button>

// セカンダリボタン
<Button variant="secondary">セカンダリ</Button>

// アクセントボタン
<Button variant="accent">アクセント</Button>

// アウトラインボタン
<Button variant="outline">アウトライン</Button>

// 透明なボタン
<Button variant="ghost">ゴースト</Button>

// リンクスタイルのボタン
<Button variant="link">リンク</Button>
```

### サイズバリエーション

```tsx
<Button size="xs">極小</Button>
<Button size="sm">小</Button>
<Button size="md">中</Button>
<Button size="lg">大</Button>
<Button size="xl">特大</Button>
```

### アイコンのみのボタン

```tsx
<Button iconOnly size="md">
  <IconComponent />
</Button>
```

### フルワイドボタン

```tsx
<Button fullWidth>幅いっぱいのボタン</Button>
```

### 角丸のカスタマイズ

```tsx
<Button rounded="none">角なし</Button>
<Button rounded="sm">小さい角丸</Button>
<Button rounded="md">標準の角丸</Button>
<Button rounded="lg">大きい角丸</Button>
<Button rounded="full">完全な丸み</Button>
```
